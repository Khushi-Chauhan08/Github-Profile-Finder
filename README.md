##GITHUB PROFILE FINDER
  -A web application that fetches and displays GitHub user data using the GitHub API.

##Features
 -Search any GitHub user
 -View profile details (name, bio, join date)
 -View top repositories
 -Battle mode: Compare two users based on stars

  -CSS
  -JavaScript
